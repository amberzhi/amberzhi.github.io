html入门学习

《html入门学习》主要介绍html的常用标签及其属性，此教程的内容主要是在开发中使用更多的内容，不能大而全，但希望浅显易懂，学习者能够快速熟悉并使用。

HTML是什么

在电脑上谷歌浏览器（效果好，没有就下载吧，下载使用谷歌浏览器是学习前端的准备工作之一）打开任意一个网页，按F12键（或者通过右键选择开发者模式或谷歌其他操作），就能看到开发者模式，选择查看Elments,就可以看到该页面的html了。先点击一下图被圈中的小图标，然后去页面点击任何一个图片、一行字之类的，你能看到右边会选中相应的html。

如果你愿意可以从网页头部开始点击，从上往下，依次看看构成网页的每一部分，看看它们各自对应的html代码。然后你会发现，页面就像一幅画那样，由一部分一部分的块状构成的。而且在html里面，有很多<div><p><span>这样的东西组成及一些链接。嗯，我们已经看清楚了html的真面目了！html全称“Hyper Text Markup Language（超文本标记语言）”，它主要由一个个描述性的标签（<div><p>）组成，也就是说我们只要记住一些常用的标签就可以了。
